
# Socionics IE Survey — Step 1 Only

This package contains a single HTML file:

- `ie_survey_step1_only.html` — interactive sliders for Ability & Value across all 8 information elements (Ne, Ni, Se, Si, Te, Ti, Fe, Fi). It computes top sociotype matches using Model A weighting and quadra values.

## How to use

### Option A — Open locally
1. Double‑click `ie_survey_step1_only.html` to open in your browser (Chrome, Safari, Firefox, Edge).
2. Move the sliders and click **Compute type**.

### Option B — Host with GitHub Pages (free)
1. Create a new public repository on GitHub (e.g., `socionics-test`).
2. Upload `ie_survey_step1_only.html` to the repository root.
3. In **Settings → Pages**, set **Source** to `main` branch and save.
4. Your link will be:
   `https://<your-username>.github.io/<repo-name>/ie_survey_step1_only.html`

### Option C — Host instantly with Netlify Drop (fastest)
1. Visit https://app.netlify.com/drop
2. Drag `ie_survey_step1_only.html` onto the page.
3. Netlify returns a public URL you can open on iPhone.
